# Emmanuel Nene Odjidja — Portfolio

Personal portfolio website for Emmanuel Nene Odjidja, M&E Specialist, Researcher, and Epidemiologist.

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion
- Deployed on Vercel

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
```
